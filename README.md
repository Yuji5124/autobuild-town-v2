# autobuild-town-v2

3D箱庭ゲーム『オートビルドタウン v2: つくりたがりの町』の企画・試作リポジトリです。

中心体験は「自分で作っていない町なのに、だんだん自分の町に見えてくる。」です。

## Phase 1 実装内容

- 静的な `index.html` / `style.css` / `main.js` 構成
- Three.js と OrbitControls をCDNから読み込む3D箱庭プロトタイプ
- タイトル画面
- 明るい空色の背景、緑色のタイル床、中央の空き地
- 仮の町として、カラフルなブロック、家、塔、ロケット、AI作業員風ロボット、住人風キャラクターを配置

## 外側から見るモード

- 斜め上から町全体を見る
- OrbitControlsで町の中心を見ながら回転できる
- ホイールでズームできる

## 主人公目線モード

- 町の中に入ったような低い視点に切り替える
- OrbitControlsは無効
- 左右キーまたはA/Dで少しだけ横移動できる
- 同じ町を「構造」ではなく「体験」として見るための確認用モード

## Phase 1.5 実装内容

- 3D箱庭の見え方を調整
- 中央の未完成土台を追加し、これから何かが作られる中心地として強調
- 仮置きブロック、足場、簡易クレーン、休憩所、変な隙間の通路を追加
- ロケットと塔のスケールを強め、主人公目線で見上げる楽しさを調整
- 外側から見るモードでは町を構造として、主人公目線では町を体験として見やすく調整

## Phase 2 実装内容

- 空からブロックが自動で落ちてくるようにした
- ブロックの雨ON/OFF、少しだけ降らす、ブロック数表示を追加
- ブロックは中央の未完成土台、ブロック置き場、町の周辺にたまる
- ブロックが増えすぎないように上限を設けた
- まだ自動建築は未実装

## Phase 3 実装内容

- 落ちたブロックにAI作業員が反応するようにした
- 作業員は、さがす・しらべる・はこぶ の状態を持つ
- 作業員ON/OFFと作業員状態表示を追加
- 吹き出しで、作業員が考えているように見える反応を追加
- まだ自動建築は行わない

## Phase 4 実装内容

- 作業員が運んだ材料が、消えるのではなく中央土台周辺の材料置き場に集まるようにした
- 中央土台の左右と奥に、台・枠・ポール・旗・板でできた材料置き場ゾーンを追加した
- 運ばれたブロックは置き場のスロットに少しだけズレや回転を入れて並び、落下中のブロックと見分けやすくした
- 「材料あつまり: 0 / 目標数」カウントを追加し、運ばれるたびに増えるようにした
- 目標数に達しても建物は完成させず、メッセージだけを出すようにした
- 材料が集まるにつれて、中央土台の周りにポール・仮の囲い・低い土台・半透明のガイド枠・旗が少しずつ増え、準備が進んで見えるようにした
- 作業員の吹き出しと画面下メッセージに、材料が集まっていく様子の文言を追加した
- 外側モードでは材料が中心に集まる様子が、主人公目線では置かれた材料や作業員の動きが近くに見えるようにした
- まだ自動建築は未実装（建物完成・自動生成・ナオシヤー・好きな場所記録・10ステップストーリー・町名保存・セーブ・物理エンジンは入れていない）

## Phase 5 予定

次のPhase 5では、集まった材料によって「小さな作りかけ構造」が少し変化する演出を検討する予定です。
