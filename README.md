# autobuild-town-v2

3D箱庭ゲーム『オートビルドタウン v2: つくりたがりの町』の企画・試作リポジトリです。

中心体験は「自分で作っていない町なのに、だんだん自分の町に見えてくる。」です。

## Phase 1 実装内容

- 静的な `index.html` / `style.css` / `main.js` 構成
- Three.js と OrbitControls をCDNから読み込む3D箱庭プロトタイプ
- タイトル画面
- 明るい空色の背景、緑色のタイル床、中央の空き地
- 仮の町として、カラフルなブロック、家、塔、ロケット、AI作業員風ロボット、住人風キャラクターを配置

## 外側から見るモード

- 斜め上から町全体を見る
- OrbitControlsで町の中心を見ながら回転できる
- ホイールでズームできる

## 主人公目線モード

- 町の中に入ったような低い視点に切り替える
- OrbitControlsは無効
- 左右キーまたはA/Dで少しだけ横移動できる
- 同じ町を「構造」ではなく「体験」として見るための確認用モード

## Phase 1.5 実装内容

- 3D箱庭の見え方を調整
- 中央の未完成土台を追加し、これから何かが作られる中心地として強調
- 仮置きブロック、足場、簡易クレーン、休憩所、変な隙間の通路を追加
- ロケットと塔のスケールを強め、主人公目線で見上げる楽しさを調整
- 外側から見るモードでは町を構造として、主人公目線では町を体験として見やすく調整

## Phase 2 実装内容

- 空からブロックが自動で落ちてくるようにした
- ブロックの雨ON/OFF、少しだけ降らす、ブロック数表示を追加
- ブロックは中央の未完成土台、ブロック置き場、町の周辺にたまる
- ブロックが増えすぎないように上限を設けた
- まだ自動建築は未実装

## Phase 3 実装内容

- 落ちたブロックにAI作業員が反応するようにした
- 作業員は、さがす・しらべる・はこぶ の状態を持つ
- 作業員ON/OFFと作業員状態表示を追加
- 吹き出しで、作業員が考えているように見える反応を追加
- まだ自動建築は行わない

## Phase 4 実装内容

- 作業員が運んだ材料が、消えるのではなく中央土台周辺の材料置き場に集まるようにした
- 中央土台の左右と奥に、台・枠・ポール・旗・板でできた材料置き場ゾーンを追加した
- 運ばれたブロックは置き場のスロットに少しだけズレや回転を入れて並び、落下中のブロックと見分けやすくした
- 「材料あつまり: 0 / 目標数」カウントを追加し、運ばれるたびに増えるようにした
- 目標数に達しても建物は完成させず、メッセージだけを出すようにした
- 材料が集まるにつれて、中央土台の周りにポール・仮の囲い・低い土台・半透明のガイド枠・旗が少しずつ増え、準備が進んで見えるようにした
- 作業員の吹き出しと画面下メッセージに、材料が集まっていく様子の文言を追加した
- 外側モードでは材料が中心に集まる様子が、主人公目線では置かれた材料や作業員の動きが近くに見えるようにした
- まだ自動建築は未実装（建物完成・自動生成・ナオシヤー・好きな場所記録・10ステップストーリー・町名保存・セーブ・物理エンジンは入れていない）

## Phase 5 実装内容

- 中央土台が、材料の集まりに合わせて少しずつ「作りかけの何か」になっていく段階演出を追加した
- 材料 3個で低い台、6個で小さな柱、9個で仮の壁、12個で足場と旗、15個で未完成の骨組みが現れる
- どの段階も完成はさせず、骨組みのまま止めている（壁も高さが不ぞろい、屋根はかぶせず半透明のガイドのみ）
- 材料が15個集まったときのメッセージを「何かになりそう。でも、まだ名前がない。」にした
- 落下ブロックは中央の作りかけ構造の上には積まず、構造の周りにリング状に集まるようにして、骨組みが見えるようにした
- 完成した家・塔・ロケットは作らず、ナオシヤー・好きな場所記録・10ステップ・名前保存も未実装のまま

## Phase 6 実装内容

- 主人公目線を「低いカメラに切り替えるだけ」から「町の中を自分で歩けるモード」にした
- W/S または ↑/↓ で前後移動できる
- A/D または ←/→ で向きを変える（旋回）ことができる
- マウスドラッグ／画面ドラッグで視点を左右に振れる
- カメラは子ども目線くらいの低い高さで、ロケット・塔・骨組み・材料置き場・作業員を近くから見られる
- 地面の範囲内に簡易クランプし、地下や空へ抜けないようにした（完全な衝突判定は未実装）
- 外側モードは OrbitControls 有効・歩行無効、主人公目線は OrbitControls 無効・歩行有効に切り替わる
- カメラリセットは外側／主人公目線のどちらでも正しく初期状態に戻る
- 主人公目線中だけ操作説明と簡易移動ボタン（前・後・左・右）を表示する
- まだ観察スポットや好きな場所記録は未実装（「見る」ボタン・好きな場所3つ・町名保存・10ステップ・ナオシヤーも未実装）

## Phase 7 実装内容

- 主人公目線で歩いて近づいた場所に反応する観察スポットを6個追加した（未完成の骨組み・材料置き場・変な隙間の通路・ロケットの足場・作業員の休憩所・見上げる塔の根元）
- 各スポットに id / name / position / radius / shortMessage / lookMessage を持たせた
- 主人公目線中は、近くにある観察スポット名（近くの場所: ○○ / なし）を表示する
- スポットの範囲に入ると、画面下メッセージを一度だけ変える（同じスポットにいる間は出し直さない）
- 主人公目線中だけ使える「ながめる」ボタンを追加し、近くのスポットの lookMessage を表示する（近くに何もなければ専用メッセージ）
- 各スポットに、薄い光のリングと小さな石の控えめな目印を置いた（収集アイテムに見えないようにした）
- 外側モードでは観察UI（場所名・ながめるボタン）を隠し、OrbitControls はこれまで通りにした
- まだ好きな場所の正式記録は未実装（好きな場所3つのカウント・町名保存・10ステップ・ナオシヤー・建物完成・セーブも未実装）

## Phase 8 予定

次のPhase 8では、見つけた場所を一時的に覚える仕組みを検討する予定です。
