# 00_core: 企画の芯

## ゲーム名

オートビルドタウン v2

## サブタイトル

つくりたがりの町

## 中心体験

自分で作っていない町なのに、だんだん自分の町に見えてくる。

## プレイヤーの役割

プレイヤーは建築者ではない。

プレイヤーは、名前のない町に呼ばれた「名づけ係」である。

町を細かく設計したり、ブロックを一つずつ置いたりする存在ではなく、勝手に作られていく町を見守り、中に入り、好きな場所を見つけ、最後に名前をつける存在である。

## 町の性質

町は勝手にブロックを集め、自分を作り続ける。

空からブロックが落ちてきて、AI作業員がそれを運び、町は自動で組み上がっていく。

町は完全に合理的でも、完全にランダムでもない。少し不器用で、少し勝手で、それでも自分なりに形になろうとする。

## ゲームの方向性

- 3D箱庭ジオラマ型ゲーム
- 空からブロックが勝手に落ちてくる
- AI作業員が勝手にブロックを運ぶ
- 町が自動で組み上がっていく
- プレイヤーは細かく建築しない
- 外から見るモードと主人公モードを切り替える
- 主人公モードでは町の中に入り、好きな場所を見つける
- 最後に町の名前を入力して保存して終わる

## 終わり方

最後に、プレイヤーは町の名前を入力する。

名前をつけた町は保存され、ゲームはそこで終わる。

この命名は単なる保存操作ではなく、名前のない町を「自分の町」として受け取るための最後の儀式である。

## ナオシヤー

ナオシヤーは、町を正しく修正しようとする存在である。

ただし、完全な悪ではない。

ナオシヤーは危ない道、曲がった構造、非効率な配置を直そうとする。ナオシヤーの言い分にも理由がある。

一方で、直しすぎると町の個性や偶然できた好きな場所が消えてしまう。

ナオシヤーは「正しさ」と「愛着」の間にある緊張を生む存在として扱う。
