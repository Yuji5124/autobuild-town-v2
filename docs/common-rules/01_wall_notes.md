# 01_wall_notes: 壁打ちメモ

このファイルは、『オートビルドタウン v2』の企画壁打ちメモを追記していく場所です。

今後の議論では、強み、弱み、危険な点、未決定事項をこのファイルに追記していく前提にします。

## 強み

- 自動で町ができていく様子そのものが見ていて楽しい
- プレイヤーが細かく建築しないため、観察、発見、愛着に集中できる
- 3D箱庭ジオラマにすることで、外から眺める楽しさと町の中に入る楽しさを両立できる
- 「自分で作っていない町なのに、だんだん自分の町に見えてくる」という中心体験が独自性になる
- 最後に町の名前をつけて保存する終わり方が、短編ゲームとして気持ちよい締めになりうる
- ナオシヤーによって、正しさ、効率、個性、愛着の対立を作れる

## 弱み

- プレイヤーが何をするゲームなのか曖昧になりやすい
- 自動建築の結果が面白くないと、鑑賞するだけの薄い体験になる
- 主人公モードでできることが弱いと、単なる確認用カメラになってしまう
- ナオシヤーの存在感が強すぎると、自動で町ができる不思議さより対決構造が目立ちすぎる
- 10ステップ構成が操作説明の列になると、物語としての気持ちよさが弱くなる

## 危険な点

- 自動化が強すぎると、プレイヤーが不要になる
- プレイヤー操作を増やしすぎると、普通の建築ゲームに近づいてしまう
- 3Dにする理由が見た目だけになると、制作負荷に対して体験の意味が弱くなる
- 町への愛着が育つ前に命名まで進むと、最後の保存が軽くなる
- ナオシヤーを単純な悪役にすると、テーマが薄くなる
- 最初からAI作業員、自動生成、主人公モード、ストーリー、敵、保存まで全部を検証しようとすると、焦点がぼやける

## 未決定事項

- プレイヤーが各ステップで具体的に何を選ぶのか
- 外から見るモードでできることの範囲
- 主人公モードで見つける「好きな場所」をどう扱うか
- 町がなぜプレイヤーを呼んだのか
- 町がなぜ自分を作り続けるのか
- ナオシヤーを敵、案内役、試練、または別の立場として扱うか
- 10ステップの各段階で、町とプレイヤーの関係がどう変化するか
- 最初のプロトタイプで何を検証するか

## 壁打ち: 主人公=名づけ係と好きな場所

町は自分を作ることはできるが、自分に名前をつけることはできない。

主人公は建築者ではなく、町の中に生まれた意味を見つけるために呼ばれた「名づけ係」である。

主人公にできることは、見ること、歩くこと、気に入ること、覚えること、最後に名前をつけること。

好きな場所は便利な場所や正しい場所ではなく、偶然できた場所、少し変な場所、外から見ると無駄や失敗に見える場所がよい。

好きな場所を3つ見つけることで、町の輪郭が見えてくる。

命名は町を完成させる操作ではなく、町の中に見つけた愛着を受け取る儀式である。

外から見るモードでは町は構造として見える。

主人公モードでは町は体験として見える。

ナオシヤーが「正しく直したい場所」と、主人公が「残したい場所」が重なる可能性がある。

今は好きな場所の判定方法やUI実装は決めない。

## 壁打ち: ナオシヤーの正しさと対立軸

ナオシヤーは悪ではない。

ナオシヤーは町を嫌っているのではなく、町を危なくしないために修正しようとしている。

ナオシヤーは町を「暮らし」ではなく「未修正の構造物」として見ている。

ナオシヤーにとって、名前のない町はまだ完成していない仮置きの構造物であり、修正対象である。

ナオシヤーが直したいものは以下の4種類に分けられる。

1. 危険
   - 崩れそう
   - 落ちそう
   - 道が細すぎる
   - 段差が大きい
   - 作業員が詰まる

2. 無駄
   - 使われていない隙間
   - 遠回りの道
   - 意味のない屋上
   - 余ったブロックの山

3. 不正
   - グリッドからずれている
   - 建物の形が変
   - 用途が未定
   - 町として分類できない

4. 未完成
   - 名前がない
   - 中心がない
   - 境界がない
   - どこが大事な場所か決まっていない

しかし主人公は同じ場所を町の中から見る。

外から見ると危険な段差でも、中に入ると町を見渡せる小さな高台かもしれない。

外から見ると無駄な隙間でも、中に入ると空がきれいに見える通路かもしれない。

外から見ると不正な行き止まりでも、主人公にとっては落ち着く場所かもしれない。

対立は、正しいか間違っているかではなく、構造として見るか、場所として感じるかである。

名前がつくことで、町は未完成物ではなく、誰かに受け取られた場所になる。

そのためナオシヤーは、名前のついた町を勝手には修正できなくなる。

ナオシヤーは最後に倒されるのではなく、少し変わる。

ラストの方向性:

「まだ直したい場所はある。けれど、そこに名前がついたなら、もう勝手には直せない。」

主人公は町を守るヒーローではなく、「直さなくてもいい理由」を見つける人である。
