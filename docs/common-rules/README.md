# docs/common-rules

このフォルダは、『オートビルドタウン v2』の企画壁打ちで使う共通ルールを保存する場所です。

実装前の企画整理、体験設計のメモ、議論中の観点、確定した方針を分けて残します。

ここに置く内容はゲーム実装とは分離します。HTML、CSS、JavaScript、Three.js、UI、プロトタイプなどの実装ファイルはこのフォルダには作りません。

## 使い分け

- `00_core.md`: 企画の芯と、変えたくない中心体験を記録する
- `01_wall_notes.md`: 壁打ちメモ、強み、弱み、危険な点、未決定事項を追記する
- `02_decisions.md`: 確定した方針だけを記録する
